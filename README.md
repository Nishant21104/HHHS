# HHHS (Helping Hands, Helping Smiles) - Charity Donation Platform

## Project Overview
HHHS (Helping Hands, Helping Smiles) is a web-based charity donation platform developed using ASP.NET Web Forms and SQL Server. 
The platform enables users to securely donate to fundraising campaigns while allowing administrators to manage and monitor donation activity efficiently.

This project was developed as part of a college academic project to demonstrate real-world implementation of a secure online donation system using the .NET Framework.

---

## Key Features

- User registration and secure login authentication
- Public campaign listing and detailed campaign view
- Integrated payment gateway for secure transactions
- Automated email confirmation after successful donation
- Donation dashboard displaying:
  - Total number of donations
  - Total amount collected
- Admin dashboard for campaign and donation management
- SQL Server database integration
- Form validation and secure data handling

---

## Project Impact
The platform promotes transparency in charitable contributions by providing structured donation tracking and administrative visibility of total funds collected.
